# CSS样式基础  

> *CSS（Cascading Style Sheets）指层叠样式表，HTML用于描述网页的内容，而CSS用于定义网页的样式，包括针对不同设备和屏幕尺寸的设计和布局。*  

> *样式定义通常保存在外部.css后缀的文件，即外部样式表文件中，通过导入该文件，只需更改文件内容即可更改整个网页的外观。*  

## CSS规则集  

CSS规则集是CSS样式表的基本组成单元，用于定义HTML元素的显示样式，由**选择器**和**声明块**两部分组成。  

例：`h1 {color:red; font-size:14px;}`，其中`h1`为选择器，指向需要设置样式的HTML元素，`{color:red; font-size:14px;}`为声明块，包含一条或多条声明，每条声明都包含一个CSS属性名称与属性值，每条声明后都要加一个分号（;）。  

> *注意：HTML语法中的属性值通常带双引号（" "），而CSS语法中属性值则无符号限制。*  

### 选择器  

CSS选择器共有五类：  

- 简单选择器  
- 组合器选择器  
- 伪类选择器  
- 伪元素选择器  
- 属性选择器  

此处讲解最基础的选择器：简单选择器，其又分为以下五种：  

- 元素选择器：根据元素名称选择HTML元素，如`p {···}`，其指向的HTML元素为`p`元素。  

- id选择器：根据HTML元素的`id`属性选择特定元素，由于元素的`id`属性具有唯一性，因此此选择器用于选择一个唯一的元素，选择时应先输入一个井号（#），后跟要选择的`id`属性值，如`#name {···}`，其指向的HTML元素为`id`属性值等于`name`的元素。  

- 类选择器：选择带有特定`class`属性的HTML元素，选择时应先输入一个句点（.），后跟要选择的`class`属性值，如`.center {···}`，其指向的元素为`class`属性值为`center`的元素。  

  若句点（.）前跟有某特定的HTML元素，则只选择带特定`class`属性的此类元素，如`p.center {···}`，其指向的元素为`class`属性值为`center`的`p`元素。  

- 通用选择器：选择页面上的所有HTML元素，格式为`* {···}`。  

- 分组选择器：如果存在多个元素具有相同的样式定义，则此选择器可对其进行分组，最大程度地缩减了代码长度，分组时应用逗号（,）分隔所选元素，如`h1, h2, p {···}`，其指向的元素为`h1`元素，`h2`元素和`p`元素。
