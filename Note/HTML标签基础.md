# HTML标签基础  

> - *HTML标签（HTML tag）是由<尖括号>包围的关键词，通常成对出现，一对标签中前者为开始标签，后者为结束标签。*
>
> - *二者中间的内容为HTML元素，元素可嵌套，即可包含其他标签。*
>
> - *括号中关键词不限制字母大小写，但最好使用小写字母。*  

## <!DOCTYPE>声明  

`<!DOCTYPE>`用于正确声明HTML版本，从而使浏览器正确显示网页内容。  

HTML5的声明为`<!DOCTYPE html>`。  

## 文档标签  

`<html>`/`</html>`标签定义了整个HTML文档的开始与结束。

## 主体标签  

`<body>`/`</body>`标签定义了HTML文档主体的开始与结束。  

## 标题标签  

与markdown语法相同，`<h1>`~`<h6>`分别定义了从大到小的六级标题。  

> *注意：搜索引擎将使用标题为网页编制索引，用户也将通过标题了解网页内容，故标题起到很重要的索引作用，应认真对待。*  

## 分隔线标签  

`<hr>`可以在HTML页面中创建分隔线。  

## 注释标签  

`<!--这是一个注释-->`  

如上面一行代码，注释标签可以提高代码可读性，类似C语言中`/*这是一个注释*/`的作用，注释标签内的内容不会显示在网页中。  

## 段落标签  

`<p>`/`</p>`可以将HTML文档分割为若干段落，两个段落之间会出现一行空白。  

## 换行标签  

`<br>`可在一个段落中间使用，在不产生一个新段落的基础上进行换行。  

> *注意：HTML文档中，浏览器会将源代码中所有连续的空格与换行合并为一个空格，故手打的换行无用。*  

## 文本格式化标签  

- `<b>`/`</b>`与`<strong>`/`</strong>`  

​	作用相同，起到加粗文本的作用，但含义不同。  

​	`<b>`定义粗体文本，`<strong>`意味着重要文本，此处用于加强语气。  

- `<i>`/`</i>`与`<em>`/`</em>`  

​	作用相同，起到倾斜文本的作用，但含义不同。

​	`<i>`定义斜体文本，`<em>`意味着重要文本，此处用于加强语气。  

- `<big>`/`</big>`与`<small>`/`</small>`  

​	前者放大文本字体，后者缩小文本字体。  

- `<sup>`/`</sup>`与`<sub>`/`</sub>`  

​	前者将元素变为文本上标，后者将元素变为文本下标。  

- `<del>`与`<ins>`  

​	前者将在文本上添加一条删除线（文本中间的一条横线，类似“划去”的意思），后者将在文本下添加一条下划线。  

## 链接标签  

HTML链接（anchor，又称超链接）是网页之间跳转的核心部分。  

HTML使用链接与网络上的其他网页或文档文件相关联，允许用户在浏览网页时通过单击链接来跳转到链接

所导航的位置。

HTML链接通过`<a href="链接内容（URL）">`/`</a>`创建，元素为链接文本，其中`<a>`起定义作用，href是一种HTML属性，起提供目标URL的作用。  

> *HTML属性为HTML元素的附加信息，一般出现在开始标签中，一种元素可附加多种属性，属性值应被包括在引号中。*  

> *注意：URL最后应始终附带一个正斜杠（/），防止服务器再进行一次添加正斜杠的HTML请求，有利于提高效率。*  

默认情况下，文本链接将以以下形式出现在网页中：  

- 未访问时：蓝色带下划线。  
- 已经访问：紫色带下划线。  
- 正在点击：红色带下划线。

> *注意：若为链接文本设置了<a href="#css">CSS</a>样式，则展示样式为CSS渲染效果。  

以下为部分其他HTML链接属性：  

- download：提示浏览器只下载链接目标，不要跳转网页。其中属性值为文件名。  
- title：属性值为鼠标光标悬停在超链接上所出现的文字。类似markdown链接语法中的超链接title。  
- id：定义在同一页面中由锚点位置跳转到另一特定位置的链接，格式为`<a href="#e.g.">`/`</a>`（锚点位置）与`<a id="e.g.">`/`</a>`（特定位置）。  

> *另有target，rel等个人感觉涉及其他非基础知识的HTML属性以及与js相关的空链接知识，此处等待后续相关领域的开发学习。*  

## 头部标签

`<head>`/`</head>`定义了HTML文档的头部区域。  

以下为部分可以添加在头部区域的标签：  

- `<title>`  

​	定义了网页在浏览器工具栏，收藏夹以及搜索引擎结果页面的标题，在HTML文档中是必需的。  

- `<base>`  

​	定义了整个HTML文档中所有链接标签的默认链接，使用href属性。  

- `<meta>`  

​	描述了网页一些基本的元数据。  

> *元数据：指网页在搜索引擎中的关键词，网页的描述内容，网页作者等不会显示在页面中，但会被浏览器解析的数据。*  

​	例：  

​	关键词：`<meta name="keywords" content="……">`  

​	描述内容：`<meta name="description" content="……">`  

​	作者：`<meta name="author" content="……">`  

​	……  

> *另有link，script等个人感觉涉及其他非基础知识的HTML标签，此处等待后续相关领域的开发学习。*  

## 样式（<a id="css">CSS</a>）标签  

CSS用于渲染HTML元素的样式。  

CSS可通过以下三种方式添加到HTML文档中：  

- 内联样式：在HTML标签中添加style属性。  
- 内部样式表：在HTML文档头部区域使用`<style>`标签。  
- 外部引用：使用外部CSS 文件。  

> *CSS最好的添加方式为外部引用，但由于目前没有系统学习CSS文件的编辑，故本次主要围绕其余两种方式展开讨论。*  

> *为简化举例，以下样式采用内联样式格式进行说明。*  

### 背景颜色  

`background-color:颜色`为背景色属性的格式，定义了一个HTML元素的背景颜色。  

例：`<p style="background-color:red">`是将本段背景色设置为红色。  

### 字体  

`font-family:字体名称`为字体属性的格式，定义了一个HTML元素中文本的字体种类（如：宋体，微软雅黑等）。  

例：`<p style="font-family:SimSun">`是将本段文本字体设置为宋体。  

### 字体颜色  

`color:颜色`为字体颜色属性的格式，定义了一个HTML元素中文本的字体颜色。  

例：`<p style="color:blue">`是将本段文本字体颜色设置为蓝色。  

### 字体大小  

`font-size:数字px`为字体大小属性的格式，定义了一个HTML元素中文本的字体型号大小。  

例：`<p style="font-size:20px">`是将本段文本字体大小设置为20。  

### 文本对齐  

`text-align:对齐方式`为文字对齐属性的格式，定义了一个HTML元素中文本的对齐方式。  

例：`<p style="text-align:center">`是将本段文本居中对齐。  

另外还有left-向左对齐，right-向右对齐等。  

> *注意：属性值都是英文！*  

如果整个文件某种标签的所有元素遵照同一CSS样式，可采用内部样式表：  

例：  

```  
<head>  
<style type="text/css">  
h2 {text-align:center;}  
p {color:yellow;}  
</style>  
</head>
```

即将整个HTML文件中的二级标题居中对齐，所有段落部分文本字体颜色设置为黄色。  

> *注意：此处type="text/css"为声明style包括的HTML元素类型为CSS，但HTML5已不需要该处声明，可省略。*  

## 图像标签  

`<img>`定义了HTML文档的图像插入，它是一个空标签，即只有属性，没有元素与结束标签。  

一般标签格式为`<img src="图像位置（URL）" alt="text">`。  

其中src属性内容为存储图像的位置链接，alt属性内容为当浏览器无法载入图像时，在原位置显示的替换文本。  

另有属性height与width可分别用于设置图像的高度与宽度，默认单位为像素，格式如下：  

`<img src="url" alt="text" height="数字" width="数字">`  

指定图像的高度与宽度可防止插入的图像破坏HTML页面的整体布局。

插入的图像默认无边框，若想添加边框可使用border属性：`border="数字"`，其属性内容为边框宽度，默认单位为像素。  

## 表格标签  

`<table>`/`</table>`定义了HTML文档的表格，HTML表格是一种用于展示结构化数据的标记语言元素。  

`<table>`包括的元素是整个HTML表格，其分为两个部分：  

- `<thead>`/`</thead>`  

​	定义表格的表头部分，所用标签为`<tr>`/`</tr>`与`<th>`/`</th>`。  

- `<tbody>`/`</tbody>`  

​	定义表格的主体部分，所用标签为`<tr>`/`</tr>`与`<td>`/`</td>`。  

其中：  

- `tr`是table row的缩写，定义表格的一行，其元素为表格某行的内容。  
- `th`是table header的缩写，定义表格的表头，其元素为表格某列或某列表头的内容。  
- `td`是table data的缩写，定义表格的数据，其元素为表格某行某列的内容。  

为了突出表格，通常我们会为表格增加边框，所用到的属性依然为border ，用法与图像标签相同，添加在`<table>`标签中。  

例：  

```
<table border="1">
  <thead>
    <tr>
      <th>列1表头</th>
      <th>列2表头</th>
      <th>列3表头</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>行1，列1</td>
      <td>行1，列2</td>
      <td>行1，列3</td>
    </tr>
    <tr>
      <td>行2，列1</td>
      <td>行2，列2</td>
      <td>行2，列3</td>
    </tr>
  </tbody>
</table>  
```

> *另有表格标题，合并单元格等内容，个人认为不算做基础，待后续补充。*  

> *注意：`th`元素文本通常加粗居中处理。*  

## 列表标签  

HTML文档支持无序、有序与自定义列表：  

- 无序列表  

​	`<ul>`/`</ul>`定义无序列表，列表项前默认为与markdown相同的黑色圆点，可使用style属性更改图案类型。

- 有序列表  

​	`<ol>`/`</ol>`定义有序列表，列表序数默认从1.开始，可使用type属性更改序数类型。  

以上两种列表的列表项标签为`<li>`/`</li>`，下面以无序列表来举例：  

```
<ol>  
<li>apple</li>  
<li>banana</li>  
</ol>  
```

- 自定义列表  

​	自定义列表为列表项与其注释的组合，`<dl>`/`</dl>`定义自定义列表。  

​	自定义列表的列表项标签为`<dt>`/`</dt>`，列表项注释标签为`<dd>`/`</dd>`。  

​	例：  

```
<dl>  
<dt>apple</dt>  
<dd>-red</dd>  
<dt>banana</dt>  
<dd>yellow</dd>  
</dl>  
```

> *注意：注释行前默认存在一个Tab的缩进。

与markdown语法相似，HTML列表中可嵌套多种元素，包括列表。  

## 区块标签  

大多数HTML元素被定义为块级元素或内联元素，块级元素在显示时通常会以新行开始，而内联元素不会。  

`<div>`/`</div>`为区块标签的一种，其元素为块级元素，无实意。  

`div`主要用于组合其他HTML元素与文档布局，它可以创建多列布局与定义文档区块。  

> *注意：`div`在CSS中可对大的内容块设置样式属性，使其更改更加便捷，具体内容待日后发掘。*  

## 表单标签  

HTML表单表示文档中的一个区域，此区域包含交互控件，用于收集用户的输入信息，并将其发送至Web服务器。  

`<form>`/`</form>`用于创建HTML表单。  

表单元素是允许用户输入内容的区域，如：文本域（textarea），下拉列表（select），单选框（radio-buttons），复选框（checkbox）等。  

`<form>`元素中最常用的标签为`<input>`，以下为`<input>`中定义输入类型的属性`type`的常用属性值：  

- 文本域——`type="text"`  

​	默认宽度20个字符，用户键入字母，数字等文本内容时可用。  

- 密码字段——`type="password"`  

​	用户键入字符将以星号或  · 显示。   

- 单选按钮——`type="radio"`  

- 复选框——`type="checkbox"`  

​	单选按钮与复选框主要区别在于能否选取多个选项，二者后都跟有`value`属性，用于定义选项含义，此时`<input>`后跟文本仅发挥解释选项的作用。  

- 提交按钮——`type="submit"`  

​	当用户单击提交按钮时，表单的内容会被发送至Web服务器。  

> *注意：`name`属性也常出现在`<input>`标签中，用于标识表单数据。*  

同时，`<form>`标签中也包含了一些重要属性：  

- `action`  

​	定义了Web服务器，属性值为URL或文件名。  

- `method`  

​	定义了表单数据的提交方式，以下为其部分属性值：  

​	-- `get`：默认值，指HTTP GET方法，表单数据将附加在`action`属性的URL上，并以 ? 作为分隔符，一般用于不敏感信息，如分页等。例：URL/?page=1，此处“page=1”就是get方法提交的数据。  

​	-- `post`：指HTTP POST方法，表单数据会包含在表单体内发送给Web服务器，一般用于敏感信息，如用户名和密码等。
